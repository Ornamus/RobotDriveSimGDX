Simulator Instructions

How to run the Power Up Simulator by Ryan Shavell from Team 1902

Oh boy, quickly writing simulator instructions that should have been written ages ago! Hopefully these are sufficient and help you get everything going. If not, well... I'll know what to change for the public release's instructions! :)

Launching and Setup


  • You launch the simulator with run.bat. If you want to change the program arguments, you change them in run.bat.

  • You need one USB controller per robot you want on the field. If you want to have drivers AND manipulators, you'll need two USB controllers per robot. To enable this, add the keyword manipulators to the program arguments in run.bat.

  • The USB controller needs to be similar in basic functionality to a Playstation or XBox controller; you'll want two joysticks, the four button thingy, a DPad, bumpers, triggers, and start/select present on the controller. If it's not a super generic XBox or Logitech controller, you'll need to map it first using the controller mapping tool, called Controller Mapping.bat. Launch it with the controller(s) you want to map plugged in, follow the instructions, close it when you're done. The controller should work in the main game now.

    • Controls


    Configuring Robots (Done in sandbox mode/before a match)
  • Start (the left button on the middle): change drive controls (options are arcade, split arcade, tank, chezy drive)
  • Select (the right button on the middle): change robot alliance color
  • DPad Right: Change robot's team number
  • DPad Down: Change robot type (different robots have different stats/abilities)

    • Playing a Robot

    Note: A blue M means that this control is available to the manipulator if there is one.

  • Right Bumper: Intake Cube (M)
  • Right Bumper: Outtake Cube (M)
  • Left Bumper: Outtake Cube from Back (not all robots have this) (M)
  • DPad Up: Eject a cube from the top alliance portal (M)
  • DPad Down: Eject a cube from the bottom alliance portal (M)
  • Activating Powerup #1 (Boost): Left button in the 4 button grid
  • Activating Powerup #2 (Force): Bottom button in the 4 button grid
  • Activating Powerup #3 (Levitate): Right button in the 4 button grid
  • Flipping An Arm: Top button in the 4 button grid (not all robots have this) (M)
  • Holding Left or Right trigger: Low Gear (robot drives slower) (not all robots have this)

    • Non-Robot Controls
  • P (keyboard): Start a match
  • I (keyboard): Cancel a Match
  • R (keyboard): Reset the field

    • Gameplay


    The game is designed to as closely mirror Power Up in all ways; during a match, point values and criteria should almost exactly match the FRC game manual. The only exceptions/differences are as follows:

  • There's no automated penalities. This means the game doesn't penalize you for, say, getting hit in your opponent's null zone.
  • The simulator hasn't been updated for Team Update 3, so the two switches won't match each other like Team Update 3 dictates (I'll be correcting this very soon)
  • There's a couple graphical issues with the Scale: robots appear to drive over it, and cubes appear to phase over it as well. This is a graphical glitch; robots who can score on the switch can place cubes onto the Scale, and robots will be able to drive "under" these cubes despite it graphically not appearing as such. Similarly, robots who can't score on the Scale can't actually place cubes onto the scale; they won't be counted and can be ran into by other robots. This will make more sense once you play and try this out for yourself.
